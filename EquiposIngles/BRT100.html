BRT-100

Personal

Descripción:

BRT-100 is waterproof (IP67) device small, light. The device is especially designed for staff in open spaces, children elderly, disabled and pets. Industrial Internal GPS module, GPRS, processor ARM, GPRS module for positioning data GPS module via text messages mobile phones sent to the authorized phone, show your exact position or free Google map or your own service monitoring platform the location directly. Meanwhile, also you can send the data to the server via GPRS, the user can log on to the server to see the playback of the track and find the position.
