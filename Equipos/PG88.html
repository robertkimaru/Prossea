PG88

Dingtek

Descripción:

El PG88 tiene dos botones de marcación rápida, hace seguimiento en tiempo real pro GPS o GPRS estación base (SMS/GPRS/TCP/UDP). Puede obtener la posición exacta por SMS, permite escuchar el sonido alrededor del PG88, permite establecer una comunicación bidireccional a través de SMS, además de que soporta reproductor de MP3 y MP4. Por sus completas funciones puede ser usado para cuidar de los adultos mayores y niños, así como propósitos como la seguridad de propiedad.

Menú de funciones
SOS Se pueden configurar más de dos números SOS para marcar automáticamente al momento de presionar el botón SOS. Esta función también sirve para enviar un mensaje en formato SMS al número configurado como 1SOS, con los datos de ubicación, así como emitir una alerta SOS a la plataforma web.
Directorio Todo número telefónico debe ser guardado directamente en la tarjeta SIM, ya que el reloj no puede agregar o editar ningún número telefónico. Use otro teléfono móvil para agregar los números a la tarjeta SIM
